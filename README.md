# Blinkit Sales Analysis Dashboard

## Overview

This project presents a Power BI dashboard built using Blinkit grocery sales data.

The dashboard provides insights into:

- Total Sales
- Average Sales
- Item Performance
- Outlet Analysis
- Customer Ratings
- Inventory Distribution

![Dashboard](Images/Dashboard.png)

## KPIs

| Metric | Value |
|----------|----------|
| Total Sales | $1.20M |
| Average Sales | $141 |
| Number of Items | 8523 |
| Average Rating | 3.9 |

## Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel
